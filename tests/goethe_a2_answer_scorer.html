Goethe-Zertifikat A2 – Antwortbogen & Selbstauswertung

Lesen & Hören: automatische Auswertung · Schreiben: Wortanzahl-Check

Die Prüfungsaufgaben (Texte, Hörtexte, Bilder) stammen aus dem offiziellen, kostenlosen Goethe-Institut A2-Modellsatz (© Goethe-Institut) und sind hier nicht enthalten — bitte bearbeiten Sie die Original-PDF-Aufgaben zuerst. Dieses Tool ist nur ein Antwortbogen zur Selbstkontrolle, ähnlich dem offiziellen Antwortbogen.

Lesen

Bearbeiten Sie zuerst die Original-Aufgaben (Lesen Teil 1–4) auf Papier oder im PDF. Tragen Sie hier nur Ihre Antworten ein.

Teil 1 (Items 1–5) — a / b / c

Teil 2 (Items 6–10) — a / b / c

Teil 3 (Items 11–15) — a / b / c

Teil 4 (Items 16–20) — a / b / c / d / e / f / x

Hören

Hören Sie zuerst die Original-Audiodateien / Transkripte. Tragen Sie hier nur Ihre Antworten ein.

Teil 1 (Items 1–5) — a / b / c

Teil 2 (Items 6–10) — a bis i

Teil 3 (Items 11–15) — a / b / c

Teil 4 (Items 16–20) — Ja / Nein

Schreiben

Schreiben Teil 1 (SMS, ca. 20–30 Wörter) und Teil 2 (E-Mail, ca. 30–40 Wörter). Fügen Sie Ihren Text hier ein — Sie erhalten eine Wortanzahl-Prüfung. Die inhaltliche Bewertung sollte eine Lehrkraft vornehmen.

Teil 1 — SMS (Ziel: 20–30 Wörter)

Teil 2 — E-Mail (Ziel: 30–40 Wörter)
